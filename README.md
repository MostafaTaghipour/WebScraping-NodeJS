# Web Scraping

Simple examples of web scrapping using node.js


## Install Packages

`npm i`


## Run It

`npm start`


### IMDB Example

`http://localhost:3000/movie/tt0111161`


### Live Score Example

`http://localhost:3000/league/football/italy/serie-a`


### Static Maps Example

`http://localhost:3000/map/51.3890/35.6892?zoom=11&width=800&height=500`
